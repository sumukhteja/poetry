# unposted letter ✉️

*an archive of echoes.*

A high-fidelity, minimalist digital poetry archive designed for a tactile and immersive reading experience.

## ✨ Features

- **Tactile Interface**: Poem entries are presented as minimalist envelopes that reveal their contents upon interaction.
- **Ambient immersion**: Optional background soundscapes to enhance the poetic atmosphere.
- **Dynamic Ink Trail**: A soft, pink ink cursor trail that follows your journey through the archives.
- **Adaptive Aesthetics**: Seamless light and dark mode integration designed for comfortable reading at any hour.
- **Performant Archive**: Fast, searchable collection of musings and field notes.

## 🎨 Design Philosophy

Built with a focus on "rich aesthetics" and "premium minimalism," *unposted letter* uses sophisticated typography (EB Garamond, Cinzel, La Belle Aurore) and subtle micro-animations to create a digital space that feels like paper and ink.

## 🛠️ Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom properties, Grid, Flexbox, Keyframe animations)
- **Logic**: Vanilla JavaScript
- **Data**: JSON-driven content for easy archival updates

## 🚀 Local Development

Simply open `index.html` in any modern web browser or use a local development server:

```bash
# Using Python
python -m http.server 8000

# Using Node (lite-server)
npx lite-server
```
